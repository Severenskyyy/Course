## Требования

- **Платформа:** 1С:Предприятие 8.3.20 или выше
- **Режим совместимости:** 8.3.20
- **Лицензия:** 1С:Предприятие 8.3 ПРОФ или выше

## Установка

1. Создайте новую информационную базу
2. Загрузите конфигурацию из файла `Configuration.cf`
3. Обновите конфигурацию базы данных
4. Запустите обработку `ИмпортКаталога.epf` для загрузки демо-данных

## Интеграция с FastAPI Backend

Конфигурация поддерживает двусторонний обмен данными с FastAPI backend:

### Экспорт в FastAPI:
- Справочник "Книги" → POST /api/import-export/books
- Справочник "Читатели" → POST /api/import-export/readers
- Регистр "Выдачи" → POST /api/import-export/loans

### Импорт из FastAPI:
- GET /api/reports/overdue/export → Отчёт "Просрочки"
- GET /api/books → Справочник "Книги"
- GET /api/readers → Справочник "Читатели"

## KPI показатели

Согласно ТЗ, система отслеживает:
- **Доля просроченных выдач** ≤ 15% (целевой порог)
- **Средняя просрочка** (дни)
- **Медианная просрочка** (дни)
- **ТОП-N читателей/книг** по просрочке
- **Коэффициент оборачиваемости** фонда
