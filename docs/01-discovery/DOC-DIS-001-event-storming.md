## Доменные события (Domain Events)

| # | Событие | Агрегат | Команда | Политика |
|---|---------|---------|---------|----------|
| E1 | КнигаДобавлена | Каталог | ДобавитьКнигу | Проверить уникальность book_code |
| E2 | КнигаОбновлена | Каталог | ОбновитьКнигу | Записать в аудит-лог |
| E3 | КнигаУдалена | Каталог | УдалитьКнигу | Проверить отсутствие активных выдач |
| E4 | ЧитательЗарегистрирован | Читатели | ДобавитьЧитателя | Проверить уникальность reader_id |
| E5 | ВыдачаОформлена | Выдачи | ОформитьВыдачу | Проверить свободность экземпляра; Записать issue_date, due_date |
| E6 | ВозвратОформлен | Выдачи | ОформитьВозврат | Рассчитать days_overdue; Освободить экземпляр |
| E7 | ПросрочкаЗафиксирована | Выдачи | — (автоматически) | today > due_date AND return_date IS NULL |
| E8 | ОтчётСформирован | Отчёты | СформироватьОтчёт | Агрегировать по фильтрам |
| E9 | CSVИмпортирован | Интеграция | ИмпортироватьCSV | Идемпотентно по PK |
| E10 | CSVЭкспортирован | Интеграция | ЭкспортироватьCSV | Формат: reader_id, book_code, days_overdue |
| E11 | КнигаЗабронирована | Бронирование | ЗабронироватьКнигу | Срок 24ч, cooldown 3 дня |
| E12 | УведомлениеОтправлено | Уведомления | — (по расписанию) | За 7/3/1 день до due_date |

## Поток событий

```
[Библиотекарь] → ДобавитьКнигу → КнигаДобавлена
[Библиотекарь] → ДобавитьЧитателя → ЧитательЗарегистрирован
[Библиотекарь] → ОформитьВыдачу → ВыдачаОформлена → [Система: проверка экз.]
[Система: таймер] → ПросрочкаЗафиксирована → УведомлениеОтправлено
[Читатель] → ВернутьКнигу → [Библиотекарь] → ОформитьВозврат → ВозвратОформлен
[Библиотекарь] → СформироватьОтчёт → ОтчётСформирован → CSVЭкспортирован
[Администратор] → ИмпортироватьCSV → CSVИмпортирован
```

## Агрегаты и ограниченные контексты

| Контекст | Агрегаты | Ответственный |
|----------|----------|---------------|
| Управление каталогом | Книга (books) | FS-UI / 1С |
| Управление читателями | Читатель (readers) | FS-UI / 1С |
| Учёт выдач | Выдача (loans) | Backend / 1С |
| Отчётность | Отчёт, KPI | Backend |
| Интеграция | CSV-обмен | Backend / 1С |
| Бронирование | Бронь (reservations) | Backend |
| Уведомления | Уведомление (notifications) | Backend |
