# User Story Map — АБИС (Кейс 8)

| Версия | Статус | Дата       |
|--------|--------|------------|
| v1.0   | Final  | 2026-03-21 |

## 1. Назначение

Отдельный документ с визуальной картой пользовательских историй, сгруппированных по эпикам и релизам. Используется для защиты на Хакатоне 1 и для приоритизации скоупа.

## 2. Карта историй

### Горизонталь: пользовательский путь библиотекаря

```
┌─ 1. Найти книгу ────┬─ 2. Обслужить читателя ─┬─ 3. Оформить выдачу ─┬─ 4. Принять возврат ─┬─ 5. Контроль ───┐
│                     │                         │                      │                      │                 │
│ US-01 Поиск по      │ US-04 Найти читателя    │ US-05 Оформить       │ US-06 Оформить       │ US-08 Отчёт     │
│ названию / автору   │ по ID / ФИО             │ выдачу               │ возврат              │ просрочек       │
│                     │                         │                      │                      │                 │
│ US-02 Фильтры       │ US-11 История выдач     │ US-07 Продлить       │ US-12 Списать       │ US-09 KPI       │
│ (жанр, год, UDK)    │ читателя                │ выдачу (Could)       │ повреждённую        │ дашборд         │
│                     │                         │                      │                      │                 │
│ US-03 Добавить      │ US-13 Уведомить         │                      │                      │ US-10 ТОП       │
│ новую книгу         │ читателя (Won’t)        │                      │                      │ должников       │
└─────────────────────┴─────────────────────────┴──────────────────────┴──────────────────────┴─────────────────┘
```

### Вертикаль: релизы

```
──────────────────────────────  MVP (релиз 1.0, защита ХК-4)  ─────────────────────────────
US-01, US-02, US-03, US-04, US-05, US-06, US-08, US-11, US-14 (импорт), US-15 (экспорт CSV)

──────────────────  Release 2 (расширение, за рамками курса)  ──────────────────

```

## 3. Связь с MoSCoW

| Приоритет | Истории                                                                | Покрытие DoD |
|-----------|------------------------------------------------------------------------|--------------|
| Must      | US-01, US-04, US-05, US-06, US-08, US-14, US-15                        | 100%         |

## 4. Связь историй с модулями системы

| История | Фронт (экран)   | Backend (эндпоинт)            | 1С (объект)              |
|---------|-----------------|-------------------------------|--------------------------|
| US-01   | CatalogPage     | `GET /books`                  | Справочник «Книги»       |
| US-04   | ReadersPage     | `GET /readers`                | Справочник «Читатели»    |
| US-05   | LoansPage       | `POST /loans`                 | Документ «ВыдачаКниги»   |
| US-06   | LoansPage       | `POST /loans/{id}/return`     | Документ «ВозвратКниги»  |
| US-08   | OverduePage     | `GET /reports/overdue`        | Отчёт «Просрочки»        |
| US-14   | IntegrationPage | `POST /import-export/*`       | Обработка импорта CSV    |
| US-15   | OverduePage     | `GET /reports/overdue/export` | Экспорт просрочек CSV    |

## 5. Вывод

Карта подтверждает, что MVP покрывает все 5 операций, указанных в RoadMap (пункт «не более пяти пользовательских операций»), и все критерии приёмки Кейса 8 покрыты хотя бы одной историей.
