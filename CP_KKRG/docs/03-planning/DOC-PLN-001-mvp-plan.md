# План разработки MVP — АБИС

Кейс 8: Библиотека — выдача и просрочки | Версия 2.0 | Дата: 13.04.2026

## Обзор

MVP реализует 5 пользовательских операций:
1. Регистрация выдачи книги
2. Оформление возврата
3. Формирование отчёта «Просрочки» с фильтрами
4. Импорт данных из CSV (books, readers, loans)
5. Экспорт отчёта в CSV

## KPI проекта

| KPI | Формула | Порог | Источник |
|---|---|---|---|
| Доля просроченных выдач | (просрочки / активные) × 100% | ≤ 15% | /api/reports/kpi |
| Скорость отчёта | Время ответа API | < 3 сек на 600 записей | Замер API |
| Идемпотентность импорта | Дубли после повторной загрузки | 0 дублей | Тест CSV дважды |
