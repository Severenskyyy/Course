# Связь экранов (UI Flow) — АБИС

Кейс 8: Библиотека — выдача и просрочки | Версия 1.0 | Дата: 12.04.2026

## Карта экранов

```
LoginPage ──► HomePage ──┬── CatalogPage ── [Карточка книги]
                         ├── ReadersPage ── [Карточка читателя]
                         ├── LoansPage ──── [Новая выдача / Возврат]
                         ├── OverduePage ── [Экспорт CSV]
                         ├── ProfilePage ── [История выдач / Уведомления]
                         ├── IntegrationPage ── [Импорт/Экспорт CSV]
                         └── SettingsPage ── [Управление пользователями]
```

## Описание экранов

| # | Экран | Компонент | Описание | Доступ |
|---|-------|-----------|----------|--------|
| 1 | Авторизация | LoginPage | JWT-логин, выбор роли | Все |
| 2 | Главная | HomePage | Дашборд KPI, быстрые действия | Все авторизованные |
| 3 | Каталог книг | CatalogPage | CRUD книг, поиск, фильтры (автор, год, UDK) | Admin, Librarian, Reader (r/o) |
| 4 | Читатели | ReadersPage | CRUD читателей, поиск по ФИО/группе | Admin, Librarian |
| 5 | Выдачи | LoansPage | Оформление выдач и возвратов, список активных | Admin, Librarian |
| 6 | Просрочки | OverduePage | Отчёт с фильтрами, экспорт CSV, KPI | Admin, Librarian |
| 7 | Профиль | ProfilePage | История выдач текущего пользователя | Все авторизованные |
| 8 | Интеграция | IntegrationPage | Импорт/экспорт CSV, обмен с 1С | Admin |
| 9 | Настройки | SettingsPage | Управление пользователями, системные параметры | Admin |

## Навигация

Основная навигация через Sidebar (боковая панель): Каталог, Читатели, Выдачи, Просрочки. Верхняя панель Navbar содержит: логотип, текущий пользователь, выход. Пункты меню отображаются в зависимости от роли.
